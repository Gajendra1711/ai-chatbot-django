# AI Course Support Chatbot (Django + OpenAI)

## 📌 Description
This is an AI-powered customer support chatbot built using Django and OpenAI API. It helps users with course details, fees, duration, and enrollment queries.

## 🚀 Features
- Real-time chatbot interaction
- Clean and responsive chat UI
- AI-generated responses using OpenAI
- Guardrails to prevent wrong answers
- Typing indicator for better UX

## 🛠 Tech Stack
- Python
- Django
- OpenAI API
- HTML, CSS, JavaScript

## ▶️ How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run server:
   python manage.py runserver
